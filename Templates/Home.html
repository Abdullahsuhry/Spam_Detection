Welcome to the Spam Message Detector

A simple and smart tool to classify your messages as Spam or Not Spam.

About Us

This project is built using Machine Learning to help users easily detect unwanted or harmful spam messages. The system uses a trained Naive Bayes model to analyze text and classify messages with high accuracy. Our goal is to provide a simple, fast, and effective spam detection experience.

Powered by AI

Instructions

  1. Click the "Predict Spam Message" button below.
  2. You will be taken to the prediction page.
  3. Enter any message you want to analyze.
  4. Click "Check" to see if the message is Spam or Not Spam.
Make sure your model and vectorizer files are loaded correctly in the backend.
